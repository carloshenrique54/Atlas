# Atlas

Gestor Corporativo Inteligente**  
Projeto escolar do curso Técnico em Desenvolvimento de Sistemas – SENAI*

Sobre o Projeto

O Atlas é um serviço web de gestão corporativa desenvolvido pela Atlas Dev com o objetivo de reduzir drasticamente o custo de mercado e a complexidade operacional para empresas e profissionais inexperientes.
Pensado para quem não quer gastar fortunas com sistemas complexos, o Atlas oferece uma interface dinâmica, intuitiva e poderosa para gerenciar o dia a dia corporativo.

Funcionalidades Principais

- Dashboard corporativo completo  
- Gerenciamento de equipes e tarefas   
- Relatórios automáticos e visualizações claras  
- Interface responsiva e moderna  
-Manipulação de Funcionarios

Tecnologias Utilizadas

- Frontend: React.js + Vite  
- Estilização: CSS    
- Backend: Node.Js
